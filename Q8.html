題目八：手風琴

請製作一個點擊按鈕可以切換文字顯示狀態的元件，有 n 多個這樣的元件，請一次選取起來並綁上監聽。

  • section1+

    accusantium1.

  • section2+

    accusantium2.

  • section3+

    accusantium3.

回答:

        const accordionItem = document.querySelectorAll(".accordion-item");
        accordionItem.forEach( item => {
            item.addEventListener("click", function () {
                //元素的部分也能使用 querySelector 的方式，這邊是使用 accordionItem 為母體(上面是用 forEach 所以這邊會是 i)去 querySelector("p") 。
                console.log(item);
                const accordionContent = item.querySelector("p");
                accordionContent.classList.toggle("active");//加入toggle class active 反之移除
            })
        });
